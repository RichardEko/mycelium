Guide — what to observe
Layer II Signal Boundary Mesh — signals flood the ring hop-by-hop; scope determines which nodes act vs which only forward.
  1. System pulse — all 16 nodes light up green. The wave spreads both clockwise and counter-clockwise from node 0; node 8 (8 hops away) receives last.
  2. Alpha group pulse — nodes 0–7 act (green); nodes 8–15 receive and forward the packet but stay dark — their boundary rejects it.
  3. Beta group pulse — only nodes 8–15 act; the alpha half stays dark despite the packet passing through them.
  4. Watch the delay column — far nodes receive the signal measurably later than near nodes; each hop adds ~50–200 ms of gossip latency.

Layer 2 — Boundary Mesh
Forwarding is unconditional
Every node forwards every gossip packet regardless of whether it is in scope. Only nodes that have joined the right group act on the signal — the boundary is purely local state. Watch the beta group (amber) during an Alpha group pulse: the packet hops through beta nodes but they do not record a receive — they only forward it onward.
// node joins its group — acts on Group signals agent.join_group("alpha"); // System scope — every node acts agent.emit("sys.pulse", SignalScope::System, payload); // Group scope — only "alpha" members act agent.emit("alpha.pulse", SignalScope::Group("alpha".into()), payload);
Epidemic propagation
Each hop adds latency
The ring topology has max 8 hops between any two nodes. With each gossip forward taking ~50–200 ms, a System pulse from node 0 takes up to ~1.6 s to reach the far side. The colour gradient encodes delay: watch the green wave spread from node 0 clockwise and counter-clockwise, meeting at node 8 last.
// subscribe — channel fires on each delivery let mut rx = agent.signal_rx("sys.pulse"); while let Some(sig) = rx.recv().await { // sig.sender, sig.scope, sig.payload record_receive_time(now_ms()); }
Scope comparison
Same packet — different reach
Fire each pulse type and compare: System pulse lights all 16 nodes. Alpha group pulse lights only nodes 0–7 (green ring half). Beta group pulse lights only nodes 8–15 (amber ring half). The gossip fan-out is identical — the difference is purely in the local boundary check each node applies on delivery.
