Layer 1 — KV Store
No central state — each node has its own view
Each agent writes only its own cell to its local KV store. The update then propagates peer-to-peer via epidemic gossip — there is no server holding ground truth. Switch to KV freshness view to see this directly: colour encodes how many ticks since each cell last changed state. A moving glider leaves a green trail; cells that haven't changed fade to amber. Enable Gossip delay and watch the write-wave ripple outward from each born or dying cell — distant nodes receiving the update flash as gossip reaches them hop by hop.
// each agent writes only its own cell agent.set("cell/3/4", alive_bytes); // agent(7,7) reads neighbour from its own KV // (gossip delivered it — may be 1 tick stale) let nb = agent.get("cell/2/4");
Layer 2 — Boundary Mesh
Forwarding is unconditional — acting is filtered
Every node forwards every signal regardless of scope. Only nodes that joined the right group act on it — the boundary is purely local state. Hover any cell: its 8 toroidal neighbours glow purple, showing exactly which nb_x_y groups this node has joined. Switch to Signal flow view to see the distinction live on every tick: purple = acting (in the group), blue = forwarding only. Click any cell to change the signal source.
// cell (3,4) joins its neighbours' groups for (nx, ny) in toroidal_neighbours(3, 4) { agent.join_group(format!("nb_{nx}_{ny}")); } // tick: System scope → every node acts agent.emit("conway.tick", SignalScope::System, payload);
Eventual consistency
The ripple is not a bug
Enable Gossip delay, then watch both views together. In KV freshness: the write-wave from a state change spreads outward — distant cells are still amber while the source is already green. In Life view: cells with an amber outline computed this generation from a neighbour snapshot that hadn't arrived yet (stale reads counter climbs). Once gossip catches up they correct. This is the two-timescale design — fast signals trigger computation; durable KV carries truth.
// per-cell tick handler reads its local view let live_count = neighbours .iter() .filter(|(nx, ny)| { agent.get("cell/{nx}/{ny}") .map(|b| b[0] == 1) .unwrap_or(false) }) .count(); // may be stale — the ripple
